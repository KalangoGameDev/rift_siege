# Rift Siege

Repositório do jogo `Rift Siege` e da infraestrutura associada de multiplayer.

## Layout

- `Scenes/`: cliente Godot do jogo.
- `rift_siege/infra/signaling-server/`: servidor de sinalização WebSocket em Go.
- `rift_siege/infra/apps/coturn/`: configuração do TURN server.
- `rift_siege/.gitea/workflows/`: deploy automatizado.
