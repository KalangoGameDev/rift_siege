# Instruções de Infra

Guia operacional para o ambiente de deploy do `Rift Siege`.

## Serviços

- Signaling Server em `:8080`
- Coturn em `:3478` e faixa de relay UDP

## Variáveis importantes

- `PSK`: chave compartilhada usada pelo servidor de sinalização
- `ADVERTISED_IP`: IP público anunciado pelo servidor
